# 1차 프로젝트 설정 및 기초 개발 완료 목록

1. 프로젝트 환경 구축
   - Expo SDK 54 기반 프로젝트 초기화 (`npx create-expo-app`)
   - TypeScript 및 Expo Router 기본 세팅 완료

2. 기본 문서화 및 정리
   - 프로젝트 시작 가이드 문서 가독성 개선 및 정리
   - 앱 실행 방법 (Xcode 시뮬레이터 기준) 가이드 작성

3. 필수 라이브러리 설치 및 설정
   - 상태 관리: **Zustand** 설치 및 샘플 스토어(`hooks/useStore.ts`) 구현
   - 환경 변수: **.env** 파일 설정 및 사용 가이드 작성

4. 라우팅 기능 구현 및 테스트
   - 신규 페이지 추가 테스트: `app/test1.tsx` 생성
   - 홈 화면(`index.tsx`)에서 신규 페이지로 이동하는 `Link` 연동 완료
   - 파일 기반 라우팅(Expo Router) 동작 확인
